# 资产看板 - 安装使用手册

一个本地运行的个人资产管理工具，数据完全存储在你自己电脑上。

## 系统要求

- Windows 10/11、macOS 或 Linux
- 硬盘空间 500MB 以上
- 需要联网（拉取行情数据）

## 一、安装 Node.js（首次需要）

访问 https://nodejs.org 下载 **LTS 版本**（v18 或更高），一路点"下一步"安装完成即可。

验证安装成功：打开命令行（Windows 按 Win+R 输入 cmd），输入：

    node -v

看到类似 `v20.11.0` 就说明装好了。

## 二、安装应用

### Windows 用户

1. 解压收到的 `asset-tracker.zip` 到任意目录（建议路径不要带中文和空格）
2. 双击 `install.bat`
3. 等待完成，第一次可能需要 3-10 分钟下载依赖

### macOS / Linux 用户

    cd asset-tracker
    ./install.sh

## 三、启动应用

### Windows

双击 `start.bat`，会自动打开浏览器进入 http://localhost:3000

### macOS / Linux

    ./start.sh

首次访问会跳转到登录页，点击"注册新账号"创建账号即可使用。

## 四、日常使用

### 启动 / 关闭

- **启动**：双击 `start.bat`（Windows）或 `./start.sh`（Mac/Linux）
- **关闭**：在命令行窗口按 `Ctrl + C`，然后关闭窗口

### 主要功能

- **首页**：总资产、累计收益、净值曲线、类别分布
- **持仓页**：添加/删除资产，输入代码或名称会自动联想
- **调仓页**：查看当前配置 vs 目标配置的偏差

### 添加资产小技巧

在"代码"框可以直接搜：

| 输入 | 效果 |
|------|------|
| `茅台` 或 `600519` | 贵州茅台（A股） |
| `苹果` 或 `AAPL` | 苹果公司（美股） |
| `腾讯` 或 `00700` | 腾讯控股（港股） |
| `btc` 或 `bitcoin` | 比特币 |
| `沪深300` | 相关ETF |

现金类资产：类别选"现金"，代码可以随便填（比如 `CASH-CNY`），数量填金额，成本价填 1。

### 生成净值快照

在持仓页点右上角"生成今日快照"，就会记录当天的总资产，用于绘制净值曲线。

**建议每天固定时间点一次**（比如收盘后）。

## 五、数据管理

- 所有数据保存在 `prisma/dev.db` 文件里
- **备份**：直接复制这个文件即可
- **迁移到新电脑**：把整个项目文件夹拷过去，`dev.db` 也一起带
- **重置**：删掉 `dev.db`，重新运行 `install.bat`

## 六、常见问题

**Q: 双击 install.bat 一闪而过？**
右键"以管理员身份运行"，或者在命令行手动进目录执行 `install.bat` 看错误信息。

**Q: 提示端口 3000 被占用？**
先关掉其他占用 3000 端口的程序，或者修改 `package.json` 里的 `"dev": "next dev -p 3001"` 换个端口。

**Q: 行情价格拉不到？**
检查网络能不能访问东方财富、CoinGecko。行情源用的是国内可访问的免费源，一般都能通。

**Q: 想让其他电脑也能访问？**
默认只能本机访问。要局域网共享，改 `package.json`：`"dev": "next dev -H 0.0.0.0"`，然后其他电脑访问 `http://你的IP:3000`。

**Q: 数据安全吗？**
数据完全存在你自己电脑上，不上传任何服务器。只有查询行情时会请求东方财富等公开数据源。
